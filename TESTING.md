# Testing

This project includes simple smoke tests for the Safety Monitoring components using Vitest + Testing Library.

To get started:

1. Install the dev dependencies:

   npm install --save-dev vitest @testing-library/react @testing-library/jest-dom jsdom

2. Run the tests:

   npm run test

Note: A `test` script was added to package.json; run `npm install` to ensure the new dev packages are installed before running tests.
